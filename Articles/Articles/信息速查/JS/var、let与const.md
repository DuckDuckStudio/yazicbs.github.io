# 三种定义变量的方式的区别

| 定义方式 | 重新赋值 | 作用域 | 变量提升 | 重复声明 |
|-----|-----|-----|-----|-----|
| var | 是 | 函数作用域 | undefined | 是 |
| let | 是 | 块级作用域 | 否 | 否 |
| const | 否 | 函数作用域 | 否 | 否 |
