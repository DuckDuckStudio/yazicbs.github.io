# JavaScript 中的注释

| 注释类型 | 方式 |
|-----|-----|
| 单行注释 | // |
| 多行注释 | /**/ |

例如:
```javascript
// 这是单行注释
/*
这是多行注释
第2行
*/
console.log(/* 这样也合法 */"这是一条输出")
```
