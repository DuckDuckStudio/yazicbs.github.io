最低系统要求

适用于芙芙工具箱 for windows

 

最低要求：

对于lite版↓

Windows 7

Python 3.7 (Tip: Python 3.9 及以上版本不支持 Windows7)

对于打包版↓

Windows 10

(打包py/pyw文件) Python 3.7 + Nuitka/Pyinstaller

如何验证配置

  • 你可以使用这个命令来验证 Python 是否正确配置：

    python --version

    一般情况下，你将会看到类似这样的输出：

    PS > python --version
    Python 3.13.3

    Python 后面带的数字代表你使用的 Python 版本。

  • 你可以使用这个命令来验证 Pyinstaller 是否正确配置：

    pyinstaller --version

    一般情况下，你将会看到类似这样的输出：

    PS > pyinstaller --version
    6.12.0

    这个数字代表你使用的 Pyinstaller 版本。

  • 你可以参考 Nuitka 文档来配置 Nuitka: Nuitka · PyPI

注意事项

  • 版本带lite的，要先安装 Python 并安装库文件后才能使用。

  • 使用lite版自带的安装库文件程序前必须先安装 Python 并正确配置环境变量。

  • 使用.bat后缀的小工具时请确保您的终端支持UTF-8编码
