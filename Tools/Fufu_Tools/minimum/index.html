最低系统要求

适用于芙芙工具箱 for windows

 

建议系统：

系统：Windows 11 x64

(lite版)Python：Python 3.12 及以上

如需使用Nuitka自动打包py/pyw文件，则最多使用 Python 3.11

终端：Windows终端

(WSA工具)其他程序：WSA以及ADB(安卓调试桥)

最低要求：

对于lite版↓

系统：Windows 7

(必需)Python：Python 3.7 (Tip:python3.9及以上版本不支持Windows7)

终端：cmd (必须支持utf-8编码)

对于打包版↓

系统：Windows 10

(打包py/pyw文件)Python：Python 3.7

终端：cmd (必须支持utf-8编码)

如何验证配置

  • 你可以使用这个命令来验证Python是否正确配置：

    python --version

    一般情况下，你将会看到类似这样的输出：

    PS C:\Users\user_name> python --version
    Python 3.12.0

    Python后面带的数字代表你使用的Python版本，使用芙芙工具箱时Python版本必须大于等于3.7.0

  • 大多数现代终端应用程序都支持 Unicode 字符，但请确保你的终端使用的字体支持 Unicode 字符。

注意事项

  • 凡是版本带lite的，必须先安装Python（3.7及以上）并安装库文件。

  • 使用lite版自带的安装库文件程序前必须先安装Python（3.7及以上）并正确配置环境变量。

  • 使用.bat后缀的小工具时请确保您的终端支持UTF-8编码

  • 如遇到无法正常打开打包版程序请尝试下载Python（3.7及以上），或者使用lite版。
