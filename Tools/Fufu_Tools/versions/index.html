版本说明

目前有lite版、打包版、安装程序、code版以及独立开发工具包

 

lite版(zip/7z)

这是一个十分轻量级的版本，占用空间最少，运行速度最快，功能与其他版本相同，但必须先配置依赖。

适用人群：

  • 能正确配置依赖[1]的

  • 需要解压即用的

打包版(zip/7z)

这是一个经过打包后的版本，占用空间较大，运行速度较慢，功能与其他版本相同，虽然有些许缺点，但你在使用这个版本时无需配置依赖。

适用人群：

  • 不配置依赖的

  • 需要解压即用的

安装程序(exe)

这是打包版的安装程序，与打包版(zip)不同的是这个版本可以实现自动新建桌面图标等功能，且自带卸载工具。占用空间较大，运行速度较慢，功能与其他版本相同，虽然有些许缺点，但你在使用这个版本时无需配置依赖。

适用人群：

  • 需要卸载程序的

  • 不配置依赖的

源码 / code版

这个版本为代码仓库中的工具的版本。使用这个版本，你可以最先体验到已知BUG修复、全新工具等，但这个版本的程序未经验证，可能会出现一些很神奇的BUG。

适用人群：

  • 想要体验到最新功能的

  • 懂得查阅文档的

  • 能正确配置依赖[1]的

如何获取：

如果您确实需要使用到这个版本，请在命令行中cd到一个目录中，然后执行git clone https://github.com/DuckDuckStudio/Fufu_Tools.git即可获取仓库源码。

在获取完仓库源码后，请记得将仓库中的cxx文件进行编译！(我使用g++编译)

独立开发工具包

这个版本为独立工具包仓库中的发行版。这个版本仅包含开发工具，同时提供预览版。

适用人群：

  • 仅需开发工具的

  • 懂得查阅文档的

如何获取：

请前往仓库发行版获取
