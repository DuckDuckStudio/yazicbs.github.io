版本说明

目前有lite版、打包版以及安装程序

 

lite版(zip)

这是一个十分轻量级的版本，占用空间最少，运行速度最快，功能与其他版本相同，但必须先配置Python。

适用人群：

懂得如何正确配置Python，或者已经正确配置且版本高于3.7[1]的人

[1] 关于芙芙工具箱的系统要求，请前往这个网站查看→[芙芙工具箱最低系统要求]

打包版(zip)

这是一个经过打包后的版本，占用空间较大，运行速度较慢，功能与其他版本相同，虽然有些许缺点，但你在使用这个版本时无需配置Python。

适用人群：

懂得一点知识，但不会配置Python环境变量等的人

不配置Python环境的人

安装程序(exe)

这是打包版的安装程序，与打包版(zip)不同的是这个版本可以实现自动新建桌面图标等功能，且自带卸载工具。占用空间较大，运行速度较慢，功能与其他版本相同，虽然有些许缺点，但你在使用这个版本时无需配置Python。

适用人群：

需要卸载程序的人

不配置Python环境的人
