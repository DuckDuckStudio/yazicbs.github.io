芙芙工具箱通用问题(Q&A)

适用于芙芙工具箱 for windows

 

Q：如何找到芙芙工具箱的安装路径

A：

你可以运行芙芙工具箱的信息确认工具来查找芙芙工具箱的安装路径。

请按照以下步骤操作：

    1. 打开芙芙工具箱

  • 2. 打开关于芙芙工具箱 > 信息确认

  • 3. 在信息确认工具的--- Directory where Fufu Tools is located ---处即可看到芙芙工具箱的安装位置
