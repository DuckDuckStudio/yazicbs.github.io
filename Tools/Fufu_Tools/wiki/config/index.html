芙芙工具箱 config.ini 文件说明

适用于芙芙工具箱 for windows

 

主要配置文件

(位于.\config.ini)

您一般不需要去修改主要配置文件

 

[实验性]休息一下配置文件

(位于.\Tools\【实验性工具】\休息一下\config.ini)

您可以通过编辑time_s的值来自定义提醒时间，默认为7200(2h)，单位为秒(s)，实际输入时不要带单位。

 

获取目录中的文件名配置文件

(位于.\Tools\文件名\config.ini)

  • Ignored file format (忽略的文件后缀)

    您可以通过编辑Ignored file format的值来自定义获取时忽略的文件后缀，以,分隔，输入时不要带.，用一个空格( )来表示不带后缀的文件。

    示例：Ignored file format = png, ,jpeg (忽略后缀名为png、jpeg与不带后缀名的文件)

  • Ignored folders (忽略的文件夹)

    您可以通过编辑Ignored folders的值来自定义获取时忽略的文件夹，以,分隔。

    示例：Ignored folders = .vs,.vscode,.git,配置文件 (忽略名为.vs、.vscode、.git与配置文件的文件夹。)

  • Specified file format (指定的文件后缀)

    您可以通过编辑Specified file format的值来自定义获取时只获取的文件后缀，以,分隔，输入时不要带.，用一个空格( )来表示不带后缀的文件。

    示例：Specified file format = png, ,jpeg (只获取后缀名为png、jpeg与不带后缀名的文件)

提醒:

Specified file format建议不要与其他参数共用，避免输出杂乱。

Specified file format的优先级最高，在定义了Specified file format时将不会执行其他规则。

 

查看电脑配置配置文件

(位于.\Tools\查看电脑配置\config.ini)

您可以在工具中对配置文件进行修改。

如果你已设置 总是显示详细信息 ，则你可以使用同目录下的fast_return_default_config.py工具快速还原默认配置。

如果你希望手动还原，则你可以手动修改配置文件中的always_open_more_info的值为False。

 

缓存清理配置文件

(位于.\Tools\缓存清理\config.ini)

  • customer_temp_folder (自定义缓存文件夹)

    你可以通过指定该项的值指定一个缓存文件夹，将值修改为False即可取消设置。

  • always_clear_ctr (总是同步清理自定义缓存文件夹)

    使用该项时，customer_temp_folder必须先指定一个值。

    你可以在程序中修改该项为True，将该项手动修改为False即可取消设置。

 

连接检测相关工具配置文件

(位于.\Tools\连接检测\config.ini)

注意：该配置文件在连接检测相关工具中通用。

  • time (自定义检测间隔)

    你可以通过指定该类下的项的值来指定检测间隔，单位为秒(s)，指定时请不要带单位。

    • wait (网络连接检测工具的间隔)

      指定该项的值来指定网络连接检测工具的检测间隔。

      默认值为10。

    • wait-github (GitHub连接检测工具的间隔)

      指定该项的值来指定GitHub连接检测工具的检测间隔。

      默认值为15。

  • url 类下的 url 项 (自定义检测链接)

    你可以通过指定该项的值来指定检测时使用的url。

    默认值为https://www.bilibili.com/

 

随机密码生成配置文件

(位于.\Tools\随机密码生成\config.ini)

您可以直接在工具中对配置文件进行修改。

注意

工具中修改时会自动忽略;，因为;在ini中被视为注释。

工具中修改时会自动转义%为%%，您不用手动转义。
