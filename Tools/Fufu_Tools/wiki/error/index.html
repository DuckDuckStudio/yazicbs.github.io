实验性工具

定时闹钟

  • ✕ 提醒时间格式不正确

    请检查你输入的时间是否为以下几种格式:

    • 时时:分分
    • 时时：分分

  • ✕ 未找到铃声文件或铃声文件不唯一

    请检查铃声文件目录 (程序所在目录下的铃声文件文件夹) 中是否有且只有一个 mp3格式的铃声文件。

    • 有且只有一个mp3格式的文件
      • 提交 Issue
    • 有不止一个mp3格式的文件
      • 你只能保留一个铃声文件
    • 没有mp3格式的文件
      • 你只能提供一个铃声文件

  • ✕ 无法查找铃声文件，请确认铃声文件文件夹是否存在

    > 在 Commit 4f9a289 前显示为 FileNotFoundError: [WinError 3] 系统找不到指定的路径。: './铃声文件'

    请检查铃声文件目录 (程序所在目录下的铃声文件文件夹) 是否存在。

    • 存在
      • 直接使用python命令执行且版本于 Commit 4f9a289 前的 → 请更新到最新版或手动cd到程序目录
      • 版本于 Commit 4f9a289 后的 → 提交 Issue
    • 不存在
      • 新建一个

搜索引擎索引检测

  • [ERROR] 不支持的搜索引擎

    请提交 Issue。

  • [ERROR] 无法连接到 {search_engine}，请检查您的网络连接

    请尝试手动访问以下对应网址，确认对应的网址确实可以正常访问。

    • 百度 → https://www.baidu.com/
    • 必应 → https://www.bing.com/ (≠https://cn.bing.com/)
    • 谷歌 → https://www.google.com/
    • 雅虎 → https://search.yahoo.com/
    如果:
    • 都能在浏览器中正常访问
      • 在终端中ping*对应的网址
      • ping*通
        • 提交 Issue
      • ping*不通
        • 请自行修复连接，我对此无法提供任何帮助
    • 无法访问
      • 请自行修复连接，我对此无法提供任何帮助

    相关文档:

    • *ping | Microsoft Learn

  • [ERROR] 检查 {search_engine} 时发生错误：{e}

    请提交 Issue。
