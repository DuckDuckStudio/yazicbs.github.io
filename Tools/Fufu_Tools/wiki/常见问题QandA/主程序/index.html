芙芙工具箱主程序使用常见问题(Q&A)

适用于芙芙工具箱 for windows

 

Q：我不知道该用什么打开lite版的主程序

A:
如果你在打开lite版的主程序时遇到下面的情况，你可能还未在你的操作系统中安装python。请下载最新版的python并配置环境变量PATH。

相关连接：

下载python

 

Q：我该如何添加python到系统PATH环境变量

Note:此方法同样适用于设置pip环境变量
A:
=> 对于还未安装python的，请在安装python时选上Add python.exe to PATH。

=> 对于已安装python的，请找到python安装路径，并复制python安装路径(例如"D:\user_name\Python\")。
然后打开控制面板(对于win10以下的版本)或者按下win+i组合键(对于win10及更高版本)打开设置。
在设置中选择 系统>系统信息>高级系统设置

在出现的窗口中选择环境变量，然后在出现的窗口中选择系统变量的Path变量，选择编辑PATH，将python安装路径的路径添加到PATH后重启设备 (可选) 即可。

检验：请在命令行(终端)中运行以下指令来测试环境变量
对于python：python --version
对于pip：pip --version
如果你看到类似这样的输出就代表配置成功，否则请尝试重启系统。

PS C:\Users\user_name> pip --version
pip 23.3.2 from D:\...\Python\Lib\site-packages\pip (python 3.12)
PS C:\Users\user_name> python --version
Python 3.12.0
PS C:\Users\user_name>

 

Q：为什么我在我的电脑上用主程序打开程序时会出现找不到路径

Note:在v1.2.8版本及以后的版本中，编码已统一为UTF-8，理应来说不会再出现这种问题。

A:

请尝试执行以下步骤：

  • 1. 请确保你的芙芙工具箱为最新版本

  • 2. 请确保你没有在非简体中文的系统中使用v1.2.7及以下版本的芙芙工具箱，如果是，请更新到最新版本

  • 3. 请确保目录及文件没有出现乱码

如果以上步骤仍然无法解决问题，请提交Issues
