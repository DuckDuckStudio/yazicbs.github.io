芙芙工具箱 开发工具 说明

适用于芙芙工具箱 独立开发工具包

 

下载zip文件的在第一次配置时，请先进行以下操作:

  1. 解压下载的 zip 文件

  2. 找到开发工具包解压目录

  3. 将该目录添加到环境变量PATH中，类似步骤请查看[Q：我该如何添加python到系统PATH环境变量] (我相信你能通过这个类似的文档看懂)

  4. 尝试在终端中使用ffdev命令，如果提示命令不存在则请重开一个终端实例/重启你的设备。

用安装程序安装的不用弄。

如果你已正确配置，你应该可以看到类似如下的输出:

                    
> ffdev
Usage: ffdev <工具> <命令> [<参数>...]
> ffdev 帮助
⚠ 无效的程序调用
[!] 可用程序: [总调用] [目录复制] [参数查重] [非UTF-8编码] [尾随空格] [需求生成] [代码行数] [账号切换] [连续push] [连续pull] [git连续尝试]

最后，使用ffdev <工具> [参数]即可执行工具，具体使用方法请参阅每个功能的--help。
