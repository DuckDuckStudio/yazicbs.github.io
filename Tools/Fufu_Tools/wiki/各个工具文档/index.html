当前可用工具

检测网络连接状况

简单的控制台连接检测工具。通过向指定的链接发送GET请求并获取返回信息和状态码来确定当前对该链接的访问状态。

查看网络密码

等效于netsh wlan show profiles后netsh wlan show profile name="{具体的网络名称}" key=clear[1]

摩斯密码编解码

对用户输入的内容进行摩斯密码编解码，具体支持字符请见芙芙工具箱 | 摩斯密码说明文件。
本工具使用 C++ 编写。

URL编解码

对用户输入的内容进行URL编解码。
解码使用urllib.parse.unquote("需要解码的内容")
编码时，先将参数分离，然后分别使用urllib.parse.quote("需要编码的内容")进行编码，最后将链接组合完整返回给用户。[2]

音频base64编解码

对指定的音频进行base64编码或对指定的内容进行base64解码为音频文件。

批量搜索

快速帮你同时搜索多个内容。[3]
您可以在工具中输入多个你想要查找的内容，然后点击“搜索”按钮就可以在默认浏览器中快速打开对应每个搜索结果的页面。
你还可以使用左下角的“导入搜索内容”功能来导入你需要搜索的内容。

爬取网站搜索结果

将搜索结果以纯文本+链接的形式显示在工具中。

打开百度

打开百度。
小白特供，忘了啥时候因为啥加的了。

系统时间显示

数字时钟。
你可以置顶它或切换深浅主题。

在线/手动自动校准系统时间

手动

等效于time。

自动

向time.windows.com[4]请求当前时间并自动校准系统时间。

自动左键/按键连点/按

帮你自动连续点按指定按键，使用F8键作为开关。

左键

使用pyautogui.click()来模拟点击输入，会time.sleep(0.01)一下来减少CPU使用率。

按键

使用pyautogui.press(具体按键)来模拟按键输入，会time.sleep(0.01)一下来减少CPU使用率。具体可用按键请参照芙芙工具箱 | [实验性]自动按键连按说明文件

连续尝试执行命令

帮你一直在指定目录下运行指定命令直到成功[5]。

定时关机

帮你在一段时间后自动关机。

按文件名/格式/特定内容查找文件

按指定的文件名/格式/含有的内容查找文件。

强制删除文件夹

尝试获取指定文件夹的权限，然后删掉它。
PS: 我没用过，不确定有没有BUG（笑
使用批处理文件(Batch)编写。

删除指定格式文件

删除指定文件夹中指定格式的所有文件。
PS: 我将此逻辑应用到了批量打包py文件工具最后的清理步骤中。

创建文件

在指定的路径生成一个指定大小、格式、文件名的占位文件。
本工具使用 C++ 编写。

文件分类

依据文件类型对文件进行初步分类。
PS: 本工具的分类依据是写死的，具体请见代码。

文件下载

下载你指定的文件。
爱来自urllib.request.urlretrieve[2]。

隐藏文件到图片

隐藏文件到图片中。
类似的可以看我之前写的小鸡习文章。

去除Python/C/Batch注释

去除指定的Python/C/Batch代码文件中的注释。
PS: 之前有写过对html代码注释的移除，但CodeQL失败。

时间/长度单位转换

帮你转化常用的时间或长度单位。

圆的参数计算

通过其中一个参数的值求出其他参数的值。

缓存清理

把系统缓存文件夹[6]中能删的文件都删掉。

随机密码生成

生成一个随机的密码。
你可以在程序中修改生成的密码的长度、字符集。并一键复制到系统剪贴板中。

随机内容展示

指定几个内容，并从中抽取一个内容展示。
你可以在程序中打开配置文件并修改随机的内容。此工具还带有随机时的动画。

视频音频提取

提取指定视频中的音频并存储到指定位置。

查看电脑配置

展示你的电脑的信息。
基本内容等效于systeminfo命令，更多信息即打开msinfo32。

进制转换

将一个进制的数转换至另一个进制。
更多信息请见芙芙工具箱 | 进制转换说明文件。
本工具有 C++ 和 Python 两个语言编写的版本，Python 版本于 1.3.7 版本从仓库中移除。

颜色代码展示

实时显示你输入的颜色代码是什么颜色。
更多信息请见芙芙工具箱 | 颜色代码展示说明文件。
你也可以在线使用这个工具。

哈希值校验

获取指定文件的哈希值。
支持 md5、sha1、sha256、sha512、blake2b、blake2s

获取目录中的文件名

依据配置文件获取指定目录中的特定文件的文件名。
支持设置忽略指定文件格式、忽略指定子文件夹、只列出指定格式。

休息一下

在指定时间后显示消息提示你休息一下。
支持设置具体时间[7]与是否开机自启动此工具。

字母大小写互换

将你指定的文本中的字母小写换大写、大写换小写。
本工具使用 C++ 编写。

英翻中

将指定的英文翻译为中文。
使用时需要网络连接。后续可能不再维护此工具。

当前可用实验性工具

自动替换指定文本

批量替换指定文件夹中指定的内容为其他内容。

Pyinstaller/Nuitka批量打包

使用Pyinstaller/Nuitka批量打包文件夹内的所有py/pyw文件。
PS: 芙芙工具箱的自动打包脚本使用相同的逻辑。

hosts修改

帮你更方便地移除/添加/列出已有的映射配置。
支持在主程序中调用和终端中调用[8]。

批量水印添加

对文件夹内的图片批量添加指定文字水印。
有关此工具的更多注意事项，请见芙芙工具箱 | [实验性]批量添加水印工具说明文件。

定时闹钟

使用自定义铃声设置闹钟。

搜索引擎索引检测

检测指定的网址是否被各个搜索引擎收录。

在线工具

可以在线使用的一些工具。
详细信息请前往芙芙工具箱 | 在线工具页面查看。
