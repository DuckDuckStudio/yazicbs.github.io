分类解释

  • [实验性]分类：还在编写但已初具雏形的程序，你可以使用他们，但可能会遇到一些未知问题。你也可以从仓库中下载这些工具来替换你使用的版本，这有可能会修复一些已知问题。这个分类下的工具可能会在之后的版本被移除这个分类，也可能被移除。
  • [测试]分类：已编写一部分但还未编写完成且未经测试或测试有问题的程序，一般不建议使用他们，因为这可能会导致一些不可逆的未知问题。这个分类下的工具可能会在之后的版本被移除。 (Note:该分类下的程序不会被添加进主程序的分类中)
  • [实验性] 分类：在本版本被移出实验性分类的工具

下载程序

lite版 打包版(zip) 安装程序(exe)
Fufu_Tools.v1.2.8-lite.zip Fufu_Tools.v1.2.8.zip Fufu_Tools_Setup.v.1.2.8.exe

版本解释

  • lite版：这是一个十分轻量级的版本，占用空间最少，运行速度最快，功能与其他版本相同，但必须先配置Python。
  • 打包版(zip)：这是一个经过打包后的版本，占用空间较大，运行速度较慢，功能与其他版本相同，虽然有些许缺点，但你在使用这个版本时无需配置Python。
  • 打包版安装程序(exe)：这是打包版的安装程序，与 打包版(zip) 不同的是这个版本 可以实现自动新建桌面图标等功能 ，且 自带卸载工具 。占用空间较大，运行速度较慢，功能与其他版本相同。

版本建议：
如果你是电脑小白，不会配置Python，请下载打包版安装程序

如果你懂得一点知识，但不会配置Python环境变量等，请下载打包版 或 打包版安装程序

如果你懂得如何正确配置Python，或者已经正确配置且版本高于3.7[1]，请下载 lite版 (对于懂得配置的强烈建议下载lite版，可以获得*更快的运行速度、更少的占用空间、更快的bug修复)*

[1] 关于芙芙工具箱的系统要求，请前往这个网站查看→ [芙芙工具箱最低系统要求]
