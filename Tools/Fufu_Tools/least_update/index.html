芙芙工具箱 v1.2.9-1

注意：lite版请下载文件名带lite的附件！
Note:压缩编码使用UTF-8
本版本为正式版！！！

该版本较上一个版本做了以下改动：

(这里的上一个版本指的是正式版v1.2.8-1)

更新

  • 修改了.gitignore(不影响使用)(v1.2.8-2)
  • 更新了SECURITY文件(不影响使用)
  • 更新了requirements.txt文件(不影响使用)(v1.2.8-2)
  • 更新了【实验性小工具说明】文件(不影响使用)(v1.2.8-2)
  • 更新了环境配置工具，使仅在[实验性]的工具中使用的库文件安装变为可选(v1.2.8-2)

修复：

  • 修复更新信息检测程序不会检测附带版本号的问题(Issues #31)
  • 修复了[测试]更新信息提示程序与[测试]更新信息提示程序（后台）的一个可能的路径问题(v1.2.8-2)
  • 修复了[测试]更新信息提示程序与[测试]更新信息提示程序（后台）无法检测预发行版版本号的问题(Issues #31)(v1.2.9)
  • 修复了[实验性]查找含有指定内容的文件工具文件夹路径中关于"与/的错误 以及 意外的结果文件输出位置的问题(v1.2.9)

新增

  • 新增了[实验性]视频音频提取工具(v1.2.8-2)
  • 新增了[实验性]自动替换指定文本工具(v1.2.8-2)
  • 新增了[实验性]查找含有指定内容的文件工具(v1.2.8-2)
  • 新增了新的版本查询py程序与c++程序(v1.2.8-2)
  • 新增了[实验性]自动使用 Pyinstaller 打包文件夹中的所有py/pyw文件工具，该工具暂未加入主程序(v1.2.8-2)

优化：

  • [实验性]自动打包所有py文件工具可以区分具体找到了多少个py文件与多少个pyw文件(Issues #37)
  • [实验性]自动打包所有py文件工具可以输出具体哪些文件炸了并自动处理"与/(Issues #35)
  • 按文件名/文件格式查找文件工具会额外检查路径是否存在(Issues #34)
  • 为查找文件分类下的三个工具添加了进度百分比显示(Issues #33)(v1.2.9)

弃用

  • 弃用了环境配置中的原bat程序(现名原bat程序（out_of_time）)(v1.2.8-2)
  • 弃用了旧的版本查询bat程序(v1.2.8-2)
