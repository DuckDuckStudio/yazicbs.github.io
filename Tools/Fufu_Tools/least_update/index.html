芙芙工具箱 v1.2.8

这个版本为芙芙工具箱的一个预览版本，皆在实现保存进度并测试的作用。
如果你是普通用户，可以选择性下载本版本或者等待v1.2.9版本发布，v1.2.9版本将会包含这个版本中可行的工具。
注意：lite版请下载文件名带lite的附件！
Note:本版本已知问题：在非 Unicode 的设备上可能会出现乱码，可以尝试使用Bandizip解压或下载7z压缩包

该版本做了以下改动：

更新：

  • 修改了.gitignore(不影响使用)
  • 更新了SECURITY文件(不影响使用)
  • 更新了requirements.txt文件(不影响使用)
  • 更新了【实验性小工具说明】文件(不影响使用)
  • 更新了环境配置工具，使仅在[实验性]的工具中使用的库文件安装变为可选
  • 重新编译了所有使用c++编写的工具

修复：

  • 修复了[测试]更新信息提示程序与[测试]更新信息提示程序（后台）的一个可能的路径问题

新增:

  • 新增了[实验性]视频音频提取工具
  • 新增了[实验性]自动替换指定文本工具
  • 新增了[实验性]查找含有指定内容的文件工具
  • 新增了新的版本查询py程序与c++程序
  • 新增了[实验性]自动使用 Pyinstaller 打包文件夹中的所有py/pyw文件工具，该工具暂未加入主程序

弃用

  • 弃用了环境配置中的原bat程序(现名原bat程序（out_of_time）)
  • 弃用了旧的版本查询bat程序
